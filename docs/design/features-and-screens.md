# RunAir — 기능 및 화면 설계서

> 최초 작성: 2026-03-28 | 버전: v1.0

---

## 1. 서비스 개요

**RunAir**는 로그인 없는 단일 페이지 웹앱으로, 현재 위치의 PM2.5 대기질을 기반으로 러너에게 오늘 몇 분 페이스로 얼마나 뛰면 되는지 알려준다.

- 타겟: 야외 러닝 전 폰으로 빠르게 확인하는 러너
- 핵심 가치: 한 화면에서 "뛰어도 되나?" + "얼마나 어떤 페이스로?" 즉시 확인

## 3. 페이스 추천 로직

### 3-1. PM2.5 기준 판정

| PM2.5 (㎍/㎥) | 등급 | 페이스 조정 | 권장 시간 |
|---|---|---|---|
| 0~15 | 좋음 (good) | 그대로 | 무제한 |
| 16~35 | 보통 (moderate) | 그대로 | 60분 |
| 36~75 | 나쁨 (bad) | +30초/km | 40분 |
| 76~150 | 매우나쁨 (veryBad) | +1분/km | 20분 |
| 151+ | 실내권장 (indoor) | — | — |

### 3-2. 온도/습도 보정 (토글 ON 시, 누적 적용)

| 조건 | 페이스 페널티 | 시간 페널티 |
|---|---|---|
| 기온 28~32도 | +15초/km | -10분 |
| 기온 33도 이상 | +30초/km | -20분 |
| 습도 70~85% | 없음 | -10분 |
| 습도 85% 이상 | +15초/km | -15분 |

> 보정 후 권장 시간이 10분 미만이면 실내 권장으로 전환

## 5. 외부 API

### 5-1. 에어코리아 (한국환경공단)

- **엔드포인트**: `getMsrstnAcctoRltmMesureDnsty` (실시간 측정소 데이터)
- **사전 처리**: `getNearbyMsrstnList`로 위/경도 → 가장 가까운 측정소 탐색
- **키 환경변수**: `AIRKOREA_API_KEY`
- **프록시 경로**: `/api/airquality?lat=&lng=`

### 5-2. 기상청 단기예보

- **엔드포인트**: `getVilageFcst` (격자 기반 단기예보)
- **현재 구현**: nx=60, ny=127 (서울 고정값, 추후 좌표 변환 필요)
- **키 환경변수**: `KMA_API_KEY`
- **프록시 경로**: `/api/weather?lat=&lng=`
